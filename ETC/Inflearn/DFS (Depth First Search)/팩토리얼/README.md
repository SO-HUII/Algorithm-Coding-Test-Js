## ✍🏻 제목 : 팩토리얼
자연수 N을 입력하면 N!값을 구하세요.

N! = n * (n-1) * (n-2) * ..... * 2 * 1입니다. 
만약 N = 5라면 5! = 5 * 4 * 3 * 2 * 1 = 120입니다.

- `입력조건` : 첫째 줄에 자연수 N(3<=n<=10)이 입력됩니다.

- `출력조건` : 첫째 줄에 N팩토리얼 값을 출력합니다.

|입력예시|출력예시|
|:------:|:----:|
|5|120|
