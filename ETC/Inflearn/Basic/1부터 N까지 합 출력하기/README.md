## ✍🏻 제목 : 1부터 N까지의 합 출력하기

자연수 N이 입력되면 1부터 N까지의 합을 출력하는 프로그램을 작성하세요. 

- `입력조건` : 첫 번째 줄에 20이하의 자연수 N이 입력된다.

- `출력조건` : 첫 번째 줄에 1부터 N까지의 합을 출력한다.


|입력예시|출력예시|
|:------:|:----:|
|6|21|


</br>

---

### 🔍 이렇게 접근 했어요 !

```javascript
for(let i = 0; i <= num; i++) {
        answer += i;
}
```
`num` : 1부터 출력하고 싶은 끝 값.

`for`문을 이용하여 1부터 num까지의 값을 차례로 반환할 값 answer에 있는 값과 더해주었다.
